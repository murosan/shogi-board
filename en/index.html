Web将棋盤
使い方

将棋の検討をブラウザで

ブラウザで動作する軽量な将棋盤 Web アプリケーションです
PWA に対応しているため、オフラインでも使用可能
macOS を完全にサポートします

ルール通りに駒を動かすことができ
王手放置・打ち歩詰め防止機能など
検討に必要な最低限の機能が付いています

将棋ソフトを用いた検討

shogi-board-server と合わせて使うことで
将棋ソフトを使って検討することができます。
もちろん macOS でも動作します

使い方は以下のドキュメントをご覧ください
- Getting Started
- 将棋ソフトを使って検討する方法

Getting Started

ダウンロード不要で試すには Web将棋盤 から
Git・Node.js を扱える方は以下のように始めることができます

$ git clone https://github.com/murosan/shogi-board.git
$ cd shogi-board
$ yarn
$ yarn start
